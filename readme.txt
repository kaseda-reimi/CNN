----mmi-eam２の内容物----------------------
nn.py       ... ニューラルネットワークの訓練用
main.py     ... mmi-eamの設計（構造探索）用
function.py ... 関数置き場
data76.txt  ... 長さ76μmの変調器のシミュレーションデータ
set-up.sh   ... GCPの環境構築用
nn_model系フォルダ  ... 学習済みニューラルネットワーク
その他       ... 最終的に使ってないので気にしなくていいです
-----------------------------------------

----nn.py--------------------------------
[各変数]
input_size = y_len*x_len
　...入力層のサイズ. y_len=6, x_len=38. function.pyからimportしてます
epochs
　...データを学習させる回数. 可変
batch_size
　...バッチサイズ（一度に学習させるデータ数） あまり検討してません. 可変
model_path = os.getcwd()+'/nn_model2'
　...学習済みニューラルネットワークの保存場所. ''内の名前を都度変える

[main()の動作]
>19-27行
訓練用データ取得
>29-37
ニューラルネットワークのモデル構築.層数などは行を足したり減らしたりで増減
>39-43
訓練
>45以降
学習結果の評価
-----------------------------------------

----main.py------------------------------
[各変数]
epochs
　... 世代交代数
group
　... 一世代の個体数

[create_first_design()]
ランダムスタートにする場合に初めの構造をデザインする関数
最終的に使っていない

[main()の動作]
>31行
スタート構造取得
>33
評価に使うニューラルネットワークを取得
>35
スタート構造を評価
>change_level(40-48)
　...元の解からどれくらい変化を加えて近傍解を生成するか（変更箇所）
探索が進むにつれて減らしています
>50-58
局所探索
